# Muks Car Rental - Premium Car Rental & Hosting Platform

A modern, fully responsive website for a premium car rental marketplace where users can rent luxury vehicles or list their own cars to earn passive income.

## 🌟 Live Demo
[View Live Website](https://codenet-technologies.github.io/muks-car-rental/)

## ✨ Features
- 🚗 Rent premium vehicles (Porsche, Tesla, Lamborghini, Mercedes, BMW)
- 💰 List your car and earn 70% of rental income
- 📱 100% responsive on all devices (mobile, tablet, desktop)
- 🔐 Owner registration system 
- 🎨 Modern glassmorphism design with custom cursor
- 🌟 Testimonials section
- 🔍 Search functionality

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6)
- Font Awesome 6
- Google Fonts (Inter)

## 📱 Responsive Breakpoints
- Desktop: 1280px+
- Tablet: 768px - 1024px
- Mobile: 320px - 768px

## 🚀 Quick Start
1. Clone the repository
2. Open `index.html` in your browser
3. No build tools required!
